My meme

This ‘crazy girl’ meme already exists but, I have never seen it with the text I attached to it.

library(magick)

#crazy_girl is the assigned name for the base image
#first annotation is for top text
#second annotation is for bottom text

crazy_girl <- image_read(path = "https://i.kym-cdn.com/entries/icons/original/000/010/496/Overly_attached_GF.jpg") %>%
  image_annotate(text = "Hey I just met you, and this is crazy..." ,
                 color = "#FFFFFF",
                 boxcolor = "black",
                 size = 80,
                 font = "Impact",
                 gravity = "north") %>%
  image_annotate(text = "but here's my number, I want your babies." ,
                 color = "#FFFFFF",
                 boxcolor = "black",
                 size = 80,
                 font = "Impact",
                 gravity = "south") 

#to turn meme into an image object

image_write(crazy_girl, "my_meme.png")

My animated GIF

This animation is a spinning square.

library(magick)

#counting vector
counts <- 1 : 6 %>% as.character()


#frame 1 is the base image (square), every additional frame refferences the base frame and rotates it as such
frame1 <- image_read("https://upload.wikimedia.org/wikipedia/commons/thumb/2/27/Red_square.svg/640px-Red_square.svg.png") %>%
  image_scale(500)

frame2 <- image_rotate(frame1, 15)

frame3 <- image_rotate(frame1, 30)

frame4 <- image_rotate(frame1, 45)

frame5 <- image_rotate(frame1, 60)

frame6 <- image_rotate(frame1, 75)


#frames in order using vector
frames <- c(frame1, frame2, frame3, frame4, frame5, frame6)

#making the animation an actual .gif file
image_animate(frames, fps = 10) %>% image_write("my_animation.gif")
